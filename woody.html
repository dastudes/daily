How to use this page
  • Your roster and stats load automatically each time you open the page.
  • Click Manage My Roster to add or drop players.
  • To add a player, type their last name in the search box and click Search. Then click the blue or green button next to their name.
  • To remove a player, click the Drop button next to their name.
  • When you're done making changes, click Save My Roster. This saves your changes so they'll still be there next time.
  • Refresh Stats reloads today's numbers from MLB without changing your roster.
  • The first time you save, you'll be asked for an access key. Ask Dave for it — you'll only need to enter it once.

About these stats

Stats are pulled live from the official MLB Stats API each time you load the page. Player names link to their Baseball Savant profiles for advanced metrics.

  • RC (Runs Created) = OBP × Total Bases
  • FIP (Fielding Independent Pitching) = ((13×HR)+(3×(BB+HBP))−(2×K))/IP + 3.10
  • PAR (Pitching Above Replacement) = (6−(FIP+ERA)/2)×IP/9

These are approximations. For more rigorous sabermetrics, see Fangraphs or Baseball Reference.
